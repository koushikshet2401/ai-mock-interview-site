# AI Mock Interview Platform

An **AI-powered mock interview platform** that simulates real technical interviews and provides automated feedback to help users improve their interview performance. The platform allows users to practice interviews, receive AI-generated evaluations, and track their progress over time.

---

# 🚀 Features

* **AI Mock Interviews** – Simulates real interview scenarios for technical roles.
* **Automated Feedback** – Generates structured feedback based on interview responses.
* **User Authentication** – Secure login and session management.
* **Interview Dashboard** – View previous interviews and track performance.
* **Transcript Analysis** – AI analyzes interview conversations to generate insights.
* **Responsive UI** – Works smoothly across desktop and mobile devices.

---

# 🧠 How It Works

1. User logs into the platform.
2. User selects a role and starts a mock interview.
3. The interview interaction is recorded as a transcript.
4. AI analyzes the transcript and evaluates performance.
5. Feedback is generated with scores and improvement suggestions.
